# # Ecotec Online – Login App (Flutter)
## Autor Oscar Guachamin
## # Nombre del proyecto
**Ecotec Online – Login App**
---
## # Descripción
Aplicación móvil desarrollada en **Flutter** que implementa un sistema básico de inicio de sesión.
La app valida credenciales locales y si son correctas redirige al usuario a una pantalla principal con una animación de transición tipo *fade*.
El proyecto aplica buenas prácticas como:
- Separación de pantallas
- Reutilización de estilos
- Navegación entre vistas
- Manejo de estados básicos
---
## # Funcionalidades
- Pantalla de Login
- Validación de usuario y contraseña
- Mensajes de error con `SnackBar`
- Navegación a pantalla principal
- Animación de transición
- Estilos reutilizables mediante `AppStyles`
